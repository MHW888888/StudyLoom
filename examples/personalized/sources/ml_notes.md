# Mini Machine Learning Notes

Gradient descent updates model parameters to reduce a loss function.

The learning rate controls the size of each update. If the learning rate is too large, training may jump around and fail to settle. If it is too small, training may be slow.

A beginner should first understand model prediction, error, loss function, gradient, and iteration before trying to implement the algorithm.

For a tiny project, implement a loop that predicts a value, computes error, updates one parameter, and prints the loss after each iteration.
