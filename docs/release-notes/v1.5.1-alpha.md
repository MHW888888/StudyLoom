# StudyLoom v1.5.1-alpha.0

This is a hardening release for the v1.5 experimental alpha extension line. It does not add major features or new platform adapters.

## Focus

- Fresh-clone validation instructions.
- Real-world fixture checklist for human alpha testing.
- Alpha risk report for unstable extension areas.
- GitHub release steps for `v1.0.0-alpha.0` and `v1.5.0-alpha.0`.
- README cleanup and optional-tool degradation notes.
- Smoke tests for template-pack quality gates and optional local-tool behavior.

## Safety Boundary

The release keeps the same boundaries:

- no cookies
- no login sessions
- no browser profiles
- no bulk platform crawling
- no paywall, DRM, anti-bot, CAPTCHA, or signature bypass
- no default platform video download

## Recommendation

Use `v1.0.0-alpha.0` as the Public Alpha baseline. Publish `v1.5.0-alpha.0` as Experimental Alpha Extensions, then use this hardening checklist before inviting wider testing.
