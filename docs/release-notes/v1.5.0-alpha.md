# StudyLoom v1.5.0-alpha.0

This alpha extension line completes the planned v1.1 through v1.5 sequence after the public alpha baseline.

## Highlights

- Real DOCX/PPTX source-fidelity intake using conservative OpenXML parsing.
- Optional source-grounded wiki and mindmap exporters.
- Hardened browser current-page extension.
- Optional local OCR, ASR, and keyframe helpers.
- Built-in template packs for exam, teacher, developer, creator, and enterprise-training workflows.

## Commands

```bash
source2study wiki build ./workspace/demo
source2study graph export ./workspace/demo --format mermaid
source2study ocr ./slide.png --output ./slide.png.ocr.txt
source2study asr inspect ./lecture.mp4
source2study keyframes inspect ./lecture.mp4
source2study templates list
```

## Safety Boundary

This release does not add high-risk platform crawling. It does not request cookies, login sessions, browser profiles, paywall bypass, DRM bypass, anti-bot bypass, or signature reverse engineering.

OCR, ASR, and keyframe extraction are local-only and optional. If Tesseract, Whisper CLI, or ffmpeg are unavailable, the commands return structured unavailable results or low-confidence placeholders.

## Known Limits

- DOCX/PPTX extraction is not full layout reconstruction.
- PPT charts are counted but not converted to structured data.
- Video keyframes are interval-based, not scene-change based yet.
- ASR alignment into EvidenceIndex is still a later workflow.
