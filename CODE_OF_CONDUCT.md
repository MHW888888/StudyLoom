# Code Of Conduct

Source2Study is built around respectful, lawful, evidence-grounded learning workflows.

## Expected Behavior

- Be kind and constructive.
- Respect privacy, copyright, and platform boundaries.
- Keep discussions focused on source fidelity, learning quality, safety, and reproducibility.
- Provide small, non-private fixtures when reporting issues.
- Disclose limitations honestly.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language.
- Requests to bypass login, paywalls, DRM, anti-bot systems, signatures, or access controls.
- Posting cookies, tokens, API keys, private documents, private course material, or sensitive local paths.
- Using the project to facilitate unauthorized scraping or redistribution.

## Enforcement

Maintainers may edit, hide, or close issues and pull requests that violate these expectations. Repeated or serious violations may lead to blocking from project spaces.
